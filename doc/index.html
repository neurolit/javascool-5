Proglet ( [String namespace] ) Static
Définit sous la forme d'un objet une Proglet pour JavaScript.

Lors de la construction de la proglet, un argument doit être passé et correpond au nom de la proglet.

La classe va tenter de charger le fichier ./proglets/< nom >/config.json. Si ce fichier existe alors il va le lire dans le but d'obtenir la configuration de base de la proglet et la liste de ses fichiers. Si le fichier n'existe pas, alors on estimera que la proglet n'est pas installé sur l'ordinateur

A la fin du chargement, la Proglet doit s'enregistrer auprès de la classe javascool.Proglets afin d'être reconnu par le reste de Java's Cool. Mais cela n'empêche pas l'utilisation de cette classe en Stand-Alone.

NB : La fonction utilisé pour le chargement est jQuery.getJSON, ce qui fait que cette classe est prête pour tourner sur un site web en ligne

code »
Parameters
namespace : String
Le nom de la proglet à chargé dans le répertoire `proglets`

Proglets ( ) Static
Gestionnaire des proglets de Java's Cool.

Cette classe permet la gestion des différentes proglets de Java's Cool. Lors de son lancement, la classe va charger le fichier `./proglets/proglets.json` dans lequel les noms des proglets installés sont écrits.

Il va lancer la création d'instance de javascool.Proglet pour tenter le chargement. C'est à la charge de la proglet de venir ensuite s'enregistrer dans le gestionnaire. Il va ensuite l'ajouter au javascool.ShortcutsPane si il est disponible.

NB : La fonction utilisé pour le chargement est jQuery.getJSON, ce qui fait que cette classe est prête pour tourner sur un site web en ligne
