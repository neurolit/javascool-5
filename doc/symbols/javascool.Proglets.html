Proglets ( ) Static
Gestionnaire des proglets de Java's Cool.

Cette classe permet la gestion des différentes proglets de Java's Cool. Lors de son lancement, la classe va charger le fichier `./proglets/proglets.json` dans lequel les noms des proglets installés sont écrits.

Il va lancer la création d'instance de javascool.Proglet pour tenter le chargement. C'est à la charge de la proglet de venir ensuite s'enregistrer dans le gestionnaire. Il va ensuite l'ajouter au javascool.ShortcutsPane si il est disponible.

NB : La fonction utilisé pour le chargement est jQuery.getJSON, ce qui fait que cette classe est prête pour tourner sur un site web en ligne
